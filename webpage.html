Anchors (i.e. links)

Achor tags create links to other pages. The URL goes in the `href` attribute. Go to DuckDockGo to search for more information

Setting `target="_blank"` will open the link in a new windo. See MDN for mor details.

Achor tags can also create links to a new location on the same page. Instead of putting a URL goes in the "href" attribute, just use the "id" of the element you want to navigate to. This allows you to Jump tp the top of the Page.

Text Emphasis

Italic

The `em` element adds extra emphasis to the text within the opening and closing tags. Ths shows up as italic text in most browsers.

Bold

You can also use the `strong` element to emphasize text. This shows up as bold text in most browsers.
